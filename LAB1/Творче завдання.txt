Придумати тест на дану тему з одним питанням і чотирма варіантами відповідей та розписати чому саме такі варіанти. Тест повинен бути цікавим і повчальним, там повинен розглядатися якийсь нюанс роботи з мовою С. (1 бал).

Що виведе наступний код?
#include <stdio.h>

int main() {
    int a = 5, b = 3;
    printf("%d\n", a++ + ++b);
    return 0;
}

1. 8
2. 9
3. 10
4. Невизначений результат.

Правильна відповідь: 9.

Пояснення варіантів відповідей:
8 (Неправильний) – Якщо припустити, що операції відбуваються в порядку зліва направо, тоді a++ повертає старе значення a, тобто 5, і інкрементує його після використання, а ++b інкрементує b перед використанням. Однак тут потрібно врахувати порядок операцій.
9 (Правильний) – Операція a++ повертає старе значення 5, потім a збільшується до 6. Операція ++b спочатку інкрементує b (тобто 3 стає 4), і потім додає до 5. Отже, результат виразу буде 5 + 4 = 9.
10 (Неправильний) – Це неправильно, оскільки сам порядок виконання операцій, де спочатку виконується a++, а потім інкрементується b, не дає результату 10.
Невизначений результат (Неправильний) – Порядок операцій у даному випадку визначений стандартом C, тому результат буде 9, а не невизначений.


Придумати одне завдання з даної теми і повністю його розібрати. Завдання повино бути цікавим і повчальним, в ньому повинен розглядатися якийсь нюанс роботи з мовою С. (1 бал)

Напишіть програму, яка зчитує літеру (A-Z або a-z) від користувача та зсуває її на задану кількість позицій вперед у межах свого регістру. Якщо зсув виходить за межі алфавіту (Z або z), він повинен повернутися на початок відповідного регістру.

Приклад коду:
#include <stdio.h>

int main() {
    char ch;
    int shift;

    printf("Введіть символ (A-Z або a-z): ");
    scanf(" %c", &ch);  // Пробіл перед %c дозволяє пропустити пробіли та нові рядки

    printf("Введіть зсув: ");
    scanf("%d", &shift);

    // Перевірка, чи введений символ є літерою
    if (ch >= 'A' && ch <= 'Z') {
        ch = 'A' + (ch - 'A' + shift) % 26;  // Збереження в межах великих літер
    } else if (ch >= 'a' && ch <= 'z') {
        ch = 'a' + (ch - 'a' + shift) % 26;  // Збереження в межах малих літер
    } else {
        printf("Помилка: введений символ не є літерою.\n");
        return 1;
    }

    printf("Новий символ: %c\n", ch);

    return 0;
}
